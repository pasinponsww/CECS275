
# CECS275 Final Project 
** Fetch from the assignment

### Objective
The goal of this project is to write a 50-year US Tax Study program in C++ using GTK framework.

### REQUIREMENTS:

* Part A:
    - A log in dialog asking users for username and password and display a successful or failure message.
    - Allow user to upload an image for their avatar.

* Part B: 
    - Select a tax year between 1975-2025
    - Once the user select a tax year, the program will show the tax bracket tables for each tax status and the standard deduction.
    - It will allow the user to input their own data and calculate their tax for that year.
    - It will also allow the user to randomly create a profile to calculate the tax.
* Part C:
    - In a separate tab, create 1000 fake tax profile cases. 
    - Reference table A-2 and calculate the percentage for each income range.
    - Provide a Pie chart to show the percentages 
    - Calculate the average income, median income, and average effective tax rate for each year from 1975 to 2025
    - Plot the your sample's average income, median income, and average effective tax rate versus the data from 1975 to 2025 using line chart.
    - You must include sufficient test cases to demonstrate your program works properly.
